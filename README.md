# Flowboard

Flowboard is a full-stack task management application designed to help users organize, track, and optimize their tasks efficiently. Beyond traditional task management, Flowboard integrates an AI-powered assistant that provides smart suggestions and insights, helping users prioritize tasks, manage time, and boost productivity.  

It is built with a modern **React frontend** and a **Flask backend**, fully deployed and ready to use.  

---

## 🌟 Features

- **User Authentication:**  
  Users can securely sign up, log in, and manage their account information. Access is restricted to authenticated users to ensure privacy and security.

- **Task Management:**  
  - Create, edit, and delete tasks easily.  
  - Mark tasks as complete or incomplete to track progress.  
  - Categorize tasks for better organization and filtering.  

- **Task Statistics:**  
  - View task completion statistics for personal insights.  
  - Track progress over time and identify patterns in task completion.  

- **AI Integration:**  
  - An AI-powered assistant provides intelligent suggestions to improve task prioritization.  
  - Generates insights based on user behavior and task trends.  
  - Helps optimize workflow by recommending which tasks to focus on next.  

- **Responsive Design:**  
  - Works seamlessly on both desktop and mobile devices.  
  - Modern, intuitive interface for smooth navigation and usability.  

---

## 🛠 How It Works

1. **Frontend (React):** Handles user interface and interacts with the backend via API calls.  
2. **Backend (Flask + SQLAlchemy):** Processes requests, manages authentication, handles database operations, and runs AI logic.  
3. **Database:** Stores user information, tasks, and stats. Production uses **PostgreSQL**, while local testing uses **SQLite**.  
4. **AI Module:** Integrated into the backend to provide real-time suggestions and insights based on user data.
